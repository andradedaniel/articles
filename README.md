## Installation

0. Checkout branch

```bash
git checkout 001-make-your-own-artisan
```

<br>

1. Install dependencies

```bash
commposer install

npm install
```

<br>

2. Try is-dead command

```bash
php is-dead inspire
```


<br>

3. Try IsDead facade

```bash
php is-dead tinker
```
```php
IsDead::call( 'inspire' );
```
