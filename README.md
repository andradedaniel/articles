## About

Access Laravel before and after running Pest tests.

<br>

## Installation

0. Checkout branch

```bash
git clone https://github.com/capsulescodes/articles.git

cd articles

git checkout 019-access-laravel-before-and-after-running-pest-tests
```

<br>

1. Install dependencies

```bash
composer install
```

<br>

2. Add environment

```bash
cp .env.example .env
```

<br>

3. Run tests

```bash
vendor/bin/pest
```
